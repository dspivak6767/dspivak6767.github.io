Unit 1: Data and Information

In Unit 1, we learned about how computers store and use information. A bit is the smallest piece of data, either a 0 or 1, and a byte is made of 8 bits. Sometimes computers have roundoff errors when they can't store a number exactly because there aren't enough bits. We also talked about analog data, which is data that changes smoothly like music or temperature, and how computers turn it into digital data. We studied lossless compression, which keeps all the original data when compressing, and lossy compression, which throws away some data to make files smaller. Finally, we learned about metadata, which is data about data, like the author or date of a file.

Unit 2: The Internet

Unit 2 focused on how computers talk to each other using networks. A computing device is any machine that can run programs, like phones or laptops. A computer network is a group of these devices connected together. Data travels along a path, which is the route it takes to get from one device to another. Devices follow a protocol, which is just an agreed set of rules so they can communicate. The internet is fault tolerant, meaning it keeps working even if some parts break. It also has scalability, which means it can grow to handle more users and devices. Bandwidth is how much data can be sent per second, and a URL is an easy-to-remember address for a website.
